# ExpressTypeScript
Develop an ExpressJS webserver using TypeScript

## Technologies
- [Node.js](https://nodejs.org/)
- [Typescript](https://www.typescriptlang.org/)

## Installation

ExpressTypeScript requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the dev server.

```sh
cd express-typescript
npm i
npm run start-dev
```

For production environments...

```sh
npm start
```
